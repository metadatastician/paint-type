== How to Update

1. **CRG Badge:** Edit the `**Current Grade:**` line in `READINESS.adoc`
2. **CI Badges:** Automatically updated by GitHub Actions
3. **Test Badges:** Updated automatically when tests pass
4. **Component Badges:** Update this file when component grades change

== Badge Generation

Run the following to generate updated badges:

[source,bash]
----
# Generate CRG badge (requires just and rsr-template-repo)
just crg-badge

# Verify CRG compliance
just crg-grade
----
