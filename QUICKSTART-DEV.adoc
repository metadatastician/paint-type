= paint-type — Quick Start for Developers
:toc:
:toclevels: 2

== Tech Stack

* **Rust** (stable) — Ephapax image core (`src/ephapax/`)
* **Idris2** — ABI type definitions and layout proofs (`src/interface/Abi/`)
* **Zig** 0.15+ — FFI bridge to C ABI (`src/interface/ffi/`)
* **AffineScript → typed-wasm** — bridge layer (stub, v0.2.0)
* **Gossamer** — linearly-typed desktop shell (v0.3.0)
* **just** — task runner

== Set Up Development Environment

=== Option A: Guix (preferred)

[source,bash]
----
guix shell
----

=== Option B: Nix (fallback)

[source,bash]
----
nix develop
----

=== Option C: Manual

Prerequisites:

* Rust stable toolchain (`rustup install stable`)
* Zig 0.15+ (`zig version` to verify)
* Idris2 (for ABI proof verification)
* `just` task runner

[source,bash]
----
git clone https://github.com/JoshuaJewell/paint-type.git
cd paint-type
just setup-dev
----

== Build

=== Zig FFI Bridge

[source,bash]
----
cd src/interface/ffi
zig build
----

=== Rust Ephapax Core

[source,bash]
----
cd src/ephapax
cargo build
----

=== Everything

[source,bash]
----
just build
----

== Project Structure

[source]
----
paint-type/
├── src/
│   ├── paint_core/         # Rust — RGBA16F tile engine (linear types)
│   ├── interface/
│   │   ├── Abi/            # Idris2 — ABI types, layout proofs, FFI wrappers
│   │   └── ffi/            # Zig — libpt C ABI bridge + integration tests
│   ├── bridges/            # AffineScript → typed-wasm bridge stubs
│   └── aspects/            # Cross-cutting: integrity, observability, security
├── verification/proofs/    # Formal proofs (Idris2, Lean4, Agda, Coq, TLA+)
├── tests/                  # Test suite
├── docs/                   # Documentation
├── .machine_readable/      # Machine-readable state (STATE, META, contractiles)
├── Justfile                # Task runner
└── 0-AI-MANIFEST.a2ml      # AI agent entry point
----

[source,bash]
----

just test           # All tests pass
just panic-scan     # No new security issues
----

== Key Invariants

Read `.machine_readable/MUST.contractile` before making changes.

Critical invariants:

* All cross-language boundaries go through the Idris2 ABI + Zig FFI — no shortcuts
* Tile memory is linearly typed — no aliased mutable tile references
* `believe_me` count must remain zero in all Idris2 files
* SPDX-License-Identifier header required on every new file
* No new TypeScript, Python, or Go files
