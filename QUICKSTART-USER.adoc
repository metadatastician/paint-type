== What is paint-type?

paint-type is a cross-platform open-source image editor in the spirit of Paint.NET —
capable enough for real work, simple enough to reach for without thinking.

**Current status**: paint-type is in early development. There is no user-installable
application yet. The first usable release (v0.3.0) will provide a basic canvas with
layers and tools running in the Gossamer desktop shell.

See link:ROADMAP.adoc[ROADMAP.adoc] for the milestone plan.

== What's Coming

* **v0.2.0**: Brush engine, compositing, undo history
* **v0.3.0**: Desktop application with layer panel, toolbar, and file I/O
* **v0.4.0**: Plugin system
* **v0.5.0**: Real-time collaboration via Burble WebRTC
* **v1.0.0**: Stable release
