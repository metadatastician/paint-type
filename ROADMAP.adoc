= paint-type Roadmap

== Current Status

Pre-alpha — v0.1.0 Foundation closed; v0.2.0 Core Image Operations is closing.
All in-repo v0.2.0 work has landed (compositing primitives, persistent
UndoGraph, basic Layer model, brush engine, pt_layer_* FFI, ABI-3/ABI-5/TP-3
proofs); E2E + fuzz + coverage CI wired and Rust CI is green on `main`. The
only remaining v0.2.0 box is the AffineScript → typed-wasm bridge — draft
`.twasm` schemas live at `src/bridges/paint-type-{tile,layer}.twasm` and now
compile with `tw build` via hardcoded IR; full general front-end → IR lowering
still gated upstream on `hyperpolymath/typed-wasm#127` (D1 codegen coverage) +
`#130` (D2 round-trip soundness), tracked from this side in
`metadatastician/paint-type#39`.
Architecture remains locked.

== Milestones

=== v0.1.0 - Foundation (closed)

* [x] Tile primitive (RGBA16F, 64×64 tiles)
* [x] Idris2 ABI definitions (`src/interface/Abi/Types.idr`, `Layout.idr`, `Foreign.idr`)
* [x] Zig FFI bridge (`src/interface/ffi/`) with integration tests
* [x] Rust Ephapax crate skeleton (`src/paint_core/`)
* [x] CI/CD pipeline (GitHub Actions, OpenSSF scorecard, SPDX compliance)
* [x] Repository governance and machine-readable metadata

=== v0.2.0 - Core Image Operations (current; tracked: #12)

* [x] Ephapax brush engine — compositing primitives **done** (PR #20: Porter-Duff over, masked blend, flatten_layer_stack); tile-level non-uniform composite **done** (PR #21); brush-stroke / kernel sampling **done** (PR #29: BrushTip soft/hard round, Brush::stamp with mask-modulated blend, Stroke point interpolation with spacing carryover); 7 additional operators **done** (PR #27: lerp, multiply, screen, in_op, out_op, atop, xor) — 11 compositing ops total
* [x] Non-destructive undo graph (persistent, O(1) branch) — PR #21
* [x] Basic layer model — new, flatten, delete, reorder — PR #23
* [x] `pt_layer_*` cross-language layer-metadata FFI — PR #28 (closes #25)
* [x] E2E pipeline test + fuzz harness + coverage reporting — PRs #32 / #33 / #34 / #35; Rust CI green on main — PRs #36 / #37 / #38
* [ ] AffineScript → typed-wasm bindings for the tile API — DRAFT `.twasm` schemas landed in `src/bridges/paint-type-{tile,layer}.twasm` (PR #40); now compiles with `tw build` via hardcoded IR; full front-end gated on `hyperpolymath/typed-wasm#127` + `#130`, tracked in paint-type#39
* [x] Formal proofs for ABI layout correctness in Idris2 — ABI-1/2/3/4/5 + TP-1/3 done (PRs #10, #19); ABI category fully closed
