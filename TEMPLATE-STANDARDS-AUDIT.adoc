== Classification

=== Authoritative Shared Standard

* External to this repo: `standards/session-management-standards/*`
* External: `standards/component-readiness-grades/*`

=== Repo-Local Binding/Integration

* `Justfile` canonical session aliases
* `session/dispatch.sh`
* `session/custom-checks.k9`
* `session/local-hooks.sh`
* `session/README.md`
* `coordination.k9`
* `AUDIT.adoc` (local gate summary)

=== Generated Runtime Artifact

* `.session/*` outputs created by local dispatcher per repository path

=== Obsolete/Duplicate/Drifted

* Template placeholder values (`rsr-template-repo`, `{{REPO}}`, `{{OWNER}}`) — now replaced throughout
* Legacy references to non-existent paths — corrected in this audit pass

== Move/Stay/Delete Guidance

=== Move Into `standards`

* Any future full protocol text drafts should move to
  `standards/session-management-standards/`.

=== Stay In `paint-type`

* Local aliases, hooks, coordination wiring, and repo-local checks.
* ABI/FFI sources, Ephapax crate, Idris2 proofs.

=== Delete/Archive

* Remove any remaining template placeholder text if discovered in future sessions.

== Maintenance Model Note

* Protocols central: maintained in `standards/session-management-standards/`.
* Policy local: maintained in this repo (`session/*.k9`, hooks, coordination bindings).
* State per-repo: generated during execution (`.session/*`) in the active working repository.
