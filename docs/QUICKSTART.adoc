= paint-type — Quickstart
:toc: preamble

Get up and running in 60 seconds.

== Prerequisites

* Git 2.40+
* `just` (command runner)
* Rust stable toolchain
* Zig 0.15+

== Clone and Build

[source,bash]
----
git clone https://github.com/JoshuaJewell/paint-type.git
cd paint-type
just build
just test
----

== Zig FFI Tests

[source,bash]
----
cd src/interface/ffi
zig build test
----

== Rust Ephapax Tests

[source,bash]
----
cd src/paint_core
cargo test
----
