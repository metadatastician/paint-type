:toc:

== Overview

paint-type is a cross-platform open-source image editor in the spirit of Paint.NET.
This document provides a high-level outline of the repository structure for quick
orientation.

For detailed architecture, see link:../TOPOLOGY.md[TOPOLOGY.md].
For the full directory guide, see link:../EXPLAINME.adoc[EXPLAINME.adoc].

== Source Code (`src/`)

[cols="1,3"]
|===
| Directory | Contents

| `src/ephapax/`
| Rust image core — RGBA16F tile engine, compositing, undo graph (linear types)

| `src/interface/Abi/`
| Idris2 ABI definitions — tile types, layout proofs, FFI wrappers (formally verified)

| `src/interface/ffi/`
| Zig FFI bridge — libpt C ABI implementation and integration tests

| `src/bridges/`
| AffineScript → typed-wasm bridge stubs (v0.2.0)

| `src/aspects/`
| Cross-cutting: integrity, observability, security aspects
|===

== Verification (`verification/`)

[cols="1,3"]
|===
| Directory | Contents

| `verification/proofs/idris2/`
| Idris2 proofs — ABI layout, tile pool invariants, pixel bounds

| `verification/proofs/lean4/`
| Lean4 proofs — API type safety, undo graph monotonicity

| `verification/proofs/agda/`
| Agda proofs — compositing totality, CRDT tile merge properties

| `verification/proofs/tlaplus/`
| TLA+ specs — plugin sandbox isolation, Burble session liveness
|===
